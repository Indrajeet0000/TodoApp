# 📝 Todo App (React)

A simple and beginner-friendly Todo App built using React. This project helps understand the core concepts of React like state management, event handling, and list rendering.

---

## 🚀 Features

* ✅ Add new tasks
* 🗑️ Delete tasks
* 📋 Display task list
* ⚡ Instant UI update using React state

---

## 🧠 Concepts Used

* React Functional Components
* useState Hook
* Event Handling
* List Rendering (map)
* Conditional Logic

---

## 📂 Project Structure

```
src/
 ├── App.js
 ├── index.js
 └── components/
```

---

## 🛠️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/Indrajeet0000/TodoApp.git
```

2. Navigate to project folder

```
cd todo-app
```

3. Install dependencies

```
npm install
```

4. Run the app

```
npm start
```

---

## 💻 Usage

* Enter a task in the input field
* Click **Add** button
* Task will appear in the list
* Click **Delete** to remove task
